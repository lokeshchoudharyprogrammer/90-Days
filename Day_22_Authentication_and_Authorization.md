# Day 22: Authentication and Authorization (AuthN/AuthZ)

## Learning Objectives
- Understand auth flows: sessions, tokens, JWT, OAuth2, OpenID Connect
- Role-based vs attribute-based access control (RBAC vs ABAC)
- Secure password storage and multi-factor authentication

## Why This Matters
Security around identity is fundamental to system correctness and user privacy.

---

## Hands-On Exercise
- Implement JWT-based authentication for a small API with login, token refresh, and protected endpoints.

## Mini Project
- Integrate OAuth2 login (e.g., GitHub) and map external identity to internal users and roles.

## Expected Outcome
You can implement secure auth flows and reason about token lifecycles and revocation.
