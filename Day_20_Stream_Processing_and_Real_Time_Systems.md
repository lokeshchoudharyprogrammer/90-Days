# Day 20: Stream Processing & Real-Time Systems

## Learning Objectives
- Learn stream processing concepts (windowing, stateful processing)
- Tools: Kafka Streams, Flink, KStreams, Spark Streaming
- Design for latency and throughput trade-offs

## Why This Matters
Stream processing enables real-time analytics, alerts, and materialized views for user-facing features.

---

## Hands-On Exercise
- Create a simple Kafka pipeline that aggregates events by user over tumbling windows.

## Mini Project
- Build a real-time leaderboard or analytics dashboard using stream processing.

## Expected Outcome
You can build stream pipelines and choose proper windowing and state management.
