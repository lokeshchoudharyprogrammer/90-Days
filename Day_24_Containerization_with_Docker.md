# Day 24: Containerization with Docker

## Learning Objectives
- Build, run, and optimize Docker images
- Understand layers, caching, and multi-stage builds
- Best practices for small, secure images

## Why This Matters
Containers standardize deployment and environment, enabling reproducible builds and scale.

---

## Hands-On Exercise
- Containerize a sample web app with a Dockerfile and run it locally.

## Mini Project
- Create multi-stage builds and optimize image size; scan images for vulnerabilities.

## Expected Outcome
You can containerize applications and produce production-ready images.
