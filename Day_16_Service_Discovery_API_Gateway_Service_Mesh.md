# Day 16: Service Discovery, API Gateway, and Service Mesh

## Learning Objectives
- Learn patterns for service discovery (DNS, client-side, server-side)
- Understand API gateways and their responsibilities
- Intro to service mesh (Sidecar proxies, Envoy, Istio)

## Why This Matters
Discovery and routing control traffic flow, observability, and security at scale.

---

## Hands-On Exercise
- Deploy a simple API gateway routing requests to two backend services (e.g., using Nginx or Traefik).

## Mini Project
- Add circuit breaking and retries via a service mesh or gateway configuration.

## Expected Outcome
You can implement service discovery and API gateway patterns and explain service mesh trade-offs.
