# Day 26: CI/CD — Pipelines, Testing, and Deployment Strategies

## Learning Objectives
- Build CI pipelines: lint, tests, build, deploy
- Understand deployment strategies: rolling, canary, blue-green, A/B
- Integrate security and compliance checks in pipeline

## Why This Matters
Automated pipelines ensure fast, reliable, and repeatable releases with quality gates.

---

## Hands-On Exercise
- Create a GitHub Actions workflow that runs tests and builds a Docker image on PR.

## Mini Project
- Extend pipeline to deploy to a staging environment and run integration tests before promotion.

## Expected Outcome
You can author CI/CD pipelines and choose appropriate deployment strategies.
